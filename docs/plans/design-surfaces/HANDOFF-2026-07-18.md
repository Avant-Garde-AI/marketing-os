- **‚öÝ env-pull caveat (learned 2026-07-18):** `vercel env pull` writes
  **sensitive-type** vars (dashboard-added secrets: PENPOT_*, MARKETING_OS_
  API_URL, MOS_PLATFORM_SERVICE_KEY, ANTHROPIC_API_KEY‚Ä¶) as EMPTY strings ‚Äî
  they are write-only but SET and working in production. Only
  **encrypted-type** vars pull faithfully. Before concluding a var is empty,
  check its `type` via GET `/v10/projects/{id}/env` ‚Äî an encrypted var that
  pulls empty is REALLY empty (that's how the piped-add victims look).
