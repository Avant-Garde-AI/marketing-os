## Logo Usage

### Primary Logo (`logo.png` / `Avant-Garde.svg`)
- **Background:** Dark navy (`#1B263B`) only
- **Size:** Minimum 32px height in UI (`h-8`); minimum 60px height in print
- **Clear space:** Equal to the height of the triangle mark on all sides
- **Never:** Recolor, stretch, add drop shadows, or place on cream/white backgrounds without a dark container

### Logo on Light Backgrounds
The primary logo is white + gold and is designed exclusively for navy backgrounds. If you need a version for light backgrounds:
- Use the SVG and swap the white fill to `#1B263B` (navy)
- Or contain the logo within a navy header/nav bar

### Favicon
Use `favicon.ico` as the browser tab icon. For modern browsers also provide the PNG variants in `/public/images/` (android-chrome, apple-touch-icon sizes).

## Reference Screenshot

`homepage-reference.png` shows the canonical implementation of this design system. Use it as the authoritative visual reference when applying the brand to new apps.

Key elements visible in the reference:
- Nav: navy bar, logo left, text links center, hamburger right
- Hero: cream background, 9xl Playfair Display headline, mixed italic, navy quote card with gold border offset
- Color palette: navy + cream + gold in correct proportions
